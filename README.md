# python-package-index

A local, static PEP 503 "simple" package index for private wheels. No server
process required for local use — `pip` can install directly from the `simple/`
directory via a `file://` URL, or you can serve it over HTTP for other
machines.

## Usage

1. Build a wheel for your project (from that project's directory):

   ```bash
   python -m build --wheel
   ```

2. Copy the resulting wheel into `wheels/`:

   ```bash
   cp dist/*.whl ~/python-package-index/wheels/
   ```

3. Regenerate the index:

   ```bash
   python ~/python-package-index/build_index.py
   ```

4. Install from it:

   ```bash
   pip install --extra-index-url file:///python-package-index/simple/ <package-name>
   ```

   Or serve it over HTTP (e.g. for other machines on the LAN):

   ```bash
   python -m http.server -d ~/python-package-index 8080
   ```

   ```bash
   pip install --extra-index-url http://<host>:8080/simple/ <package-name>
   ```

`simple/` is fully generated by `build_index.py` — don't hand-edit it, just
re-run the script after adding/removing files in `wheels/`.
